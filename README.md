# 生き延びろ！こうかとん！

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
キャラクター（こうかとん）を動かし敵を倒すゲーム

## ゲームの実装
### 共通基本機能
* こうかとんに関するクラス
* ビームに関するクラス
* 敵に関するクラス
* 敵撃破に関するクラス

### 担当追加機能
* ボスに関するクラス
* ボスの攻撃に関すクラス
